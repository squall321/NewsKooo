# NewsKoo Frontend

React 18 + TypeScript + Vite SPA. Scaffolded in **Phase 8**.

Planned stack: Tailwind CSS + shadcn/ui, TanStack Query + Table, Recharts/visx,
Zustand, React Router. Talks to the FastAPI backend (REST + SSE/WebSocket).

Views: Dashboard (ingestion health + volume), Search (FTS + semantic), Trends
(topic/entity velocity charts), Issue alerts feed, Report viewer, Source admin.

```bash
npm install
npm run dev      # http://localhost:5173
```
